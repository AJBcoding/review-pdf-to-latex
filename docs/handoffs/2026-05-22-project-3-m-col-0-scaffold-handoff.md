## Gotchas worth carrying forward

- **T3 commit identity** — memory note saved at `~/.claude-accounts/.../memory/t3-commit-identity.md`. The GAS TOWN shell env overrides git user.name in t3 (same issue as agent-viewer). Prefix commits with `GIT_AUTHOR_NAME=AJBcoding GIT_COMMITTER_NAME=AJBcoding GIT_AUTHOR_EMAIL=150540200+AJBcoding@users.noreply.github.com GIT_COMMITTER_EMAIL=...`. The first attempt this session tagged the commit `review_pdf_to_latex/crew/anthony` and had to be reset --soft + re-committed.
- **TanStack Router validateSearch pollution** — adding new search params to one route can break navigate-callback typing in sibling routes that share a parent layout. The `stripDiffSearchParams` helper in `~/t3/apps/web/src/diffRouteSearch.ts` is now responsible for dropping any newly added column-route search params. If you add more, extend that helper.
- **react-resizable-panels v4 API** — `PanelGroup` → `Group`, `PanelResizeHandle` → `Separator`, `direction` → `orientation`. The v3 API will give "no exported member" errors.
- **Route tree regen** — `routeTree.gen.ts` is autogenerated by `@tanstack/router-plugin` during `vite dev`. After adding a new route file, run `bun run dev` briefly to regen, then typecheck.
